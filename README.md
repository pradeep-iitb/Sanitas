# Sanitas

Prototype full-stack app for a health chatbot using Google Gemini (stubbed by default).

Project structure:

```
project/
 ├── frontend/
 │    ├── index.html
 │    ├── styles/
 │    └── scripts/
 └── backend/
	├── server.js
	├── routes/
	├── controllers/
	├── models/
	├── config/
	└── utils/
```

See `backend/.env.example` for required environment variables.
